4. GUI

A interface gráfica, ou GUI (Graphic User Interface), é a forma visual de interação com o sistema operacional. É aquilo que permite ao usuário usar o computador por meio de janelas, ícones, menus, botôes, barras e ponteiro do mouse, em vez de depender apenas de comandos de texto.

Ela existe para tornar a interação com o sistema mais intuitiva. Em vez de lembrar o nome exato de um comando para abrir um programa ou acessar uma pasta, o usuário pode clicar em elementos visuais que representam essas ações.

No linux, a interface gráfica tem uma característica muito importante: ela é mais separada do restante do sistema do que vocês pensam. O sistema pode funcionar sem interface gráfica, mas existem várias interfaces gráficas diferentes. A interface sozinha não define a identidade do sistema.

No Windows, o usuário comum tende a perceber o sistema como um pacote mais fechado, mais unificado. No Linux, há mais modularidade, o ambiente gráfico pode ser trocado, alterado ou até nem existir.

No Linux, não falamos apenas em "a interface gráfica" mas em ambientes gráficos, desktop environments e window managers.

Alguns exemplos:
- GNOME
- KDE Plasma
- XFCE
- LXQt
- Cinnamon
- MATE

Todos cumprem uma função parecida: oferecer ao usuário uma experiência visual para interagir com o sistema. Mas eles mudam em aparência, ferramentas, consumo de recursos e filosofia de uso.
